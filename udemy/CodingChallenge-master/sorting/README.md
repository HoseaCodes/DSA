# Sorting Algorithms

| Algorithm      |   Time   | Space |
| -------------- | :------: | ----: |
| Bubble Sort    |  O(n^2)  |  O(1) |
| Selection Sort |  O(n^2)  |  O(1) |
| Insertion Sort |  O(n^2)  |  O(1) |
| Merge Sort     | O(nlogn) |  O(n) |
